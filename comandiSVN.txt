Potreste non avere installato nel vostro sistema il programma SVN che gestisce questo tipo di servizi, se non lo avete dovete eseguire nella console:

sudo apt-get install subversion

Comandi SVN:

svn checkout https://elanja-project.googlecode.com/svn/trunk/ elanja-project --username <vostroUserName>		CHECKOUT

svn checkout https://elanja-project.googlecode.com/svn/trunk/ elanja-project --username zissex			"per Jacopo"

svn checkout https://elanja-project.googlecode.com/svn/trunk/ elanja-project --username mrblasco                      "per Andrea"

svn checkout https://elanja-project.googlecode.com/svn/trunk/ elanja-project --username elisaomodei@yahoo.com	"per Elisa"

una volta eseguito questo comando vi chiede la password che la dovete prendere dalla pagina del progetto:

Home -> Source -> Checkout -> googlecode.com password

cliccando sul link vi si apre un altra pagina con la vostra password, la inserite nella console e poi vi fa il checkout.

Il  checkout non fa altro che scaricare tutti i file che sono stati modificati rispetto alla versione che voi avete nel vostro pc.

svn add <file>		aggiunge il file specificato alla svn

svn remove <file>		rimuove il file specificato dalla svn

svn update			fa un aggiornamento dei file, solo se avete gia una versione precedente del progetto nel vostro pc

svn ci				esegue un commit, cioe' invia quello che voi avete modificato del progetto alla svn cosi' che gli altri possano fare update delle modifiche

dopo che avete eseguito svn ci, vi cheiede di scrivere un commento per descrivere le modifiche che avete fatto, e' buona norma scrivere un commento
esplicativo, cosi' che gli altri possano rendersi conto subito delle modifiche apportate.
