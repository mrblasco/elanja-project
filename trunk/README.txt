Necessarie librerie fltk >= 1.3
*******************************************************************
########### Ambiente UNIX ###########

Compilazione:

andare nella cartella principale del progetto:

	cd ~/elanja-project

ed eseguire:

	make textual

per compilare il progetto in modalita' testuale;

	make graphic

per compilare il progetto in modalita' grafica.

Una volta compilato il tutto, potete eseguire il tool invocando:

	./elanja
